# python-repo-template

Demonstrating package structure.

```
repo_name/
│
├── src/
│   ├── package1/
│   │   ├── sub_package1
│   │   ├── sub_package2
│   │   ├── __init__.py
│   ├── package2
│   ├── main.py
│   └── run.py
│
├── deployment/
├── docs/
├── scripts/
├── tests/
│   ├── test_helloworld.py
│   └── test_helpers.py
│
├── dist/
│
├── .gitignore
├── .gitlab-ci.yml
├── LICENSE
├── README.md
├── requirements.txt
└── setup.py
```
